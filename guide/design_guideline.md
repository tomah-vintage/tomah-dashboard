# Email Marketing Platform Design Guidelines

## Brand Identity

### Logo & Navigation

- **Logo**: Clean, minimalist curved design (appears to be stylized "A" or wave)
- **Color**: Dark charcoal/black with white accent
- **Navigation**: Horizontal, clean typography with orange accent on active state
- **CTA Buttons**: Rounded corners, white background with dark text for secondary, orange for primary

### Color Palette

#### Primary Colors

- **Orange/Coral**: `#FF6B35` (Primary CTAs, accents, highlights)
- **Dark Charcoal**: `#2C2C2C` (Text, navigation, logo)
- **Light Gray**: `#F8F9FA` (Background, cards)
- **White**: `#FFFFFF` (Cards, button backgrounds)

#### Secondary Colors

- **Yellow**: `#FFD700` (Accent elements, icons)
- **Light Blue**: `#E3F2FD` (Data visualization backgrounds)
- **Purple**: `#9C27B0` (Chart accents, secondary data)
- **Green**: `#4CAF50` (Positive metrics, success states)

#### Status Colors

- **Success**: `#4CAF50` (Growth indicators, positive metrics)
- **Warning**: `#FF9800` (Attention states)
- **Error**: `#F44336` (Negative metrics, errors)
- **Info**: `#2196F3` (Informational elements)

## Typography

### Font Hierarchy

- **Primary Font**: Sans-serif (appears to be Inter or similar)
- **Headings**: Bold, large scale for hero text
- **Body Text**: Regular weight, high contrast
- **Small Text**: Lighter weight for secondary information

### Scale & Weights

- **Hero Title**: 48px+, Bold (700)
- **Section Headers**: 32px, Bold (700)
- **Body Text**: 16px, Regular (400)
- **Small Text**: 14px, Regular (400)
- **Micro Text**: 12px, Light (300)

## Layout System

### Grid Structure

- **Container**: Max-width with centered alignment
- **Columns**: Flexible grid system with consistent gutters
- **Spacing**: 24px base unit for vertical rhythm
- **Padding**: 16px minimum for cards, 24px for sections

### Component Spacing

- **Section Margins**: 80px+ between major sections
- **Card Padding**: 24px internal padding
- **Button Padding**: 12px vertical, 24px horizontal
- **Icon Spacing**: 8px from text elements

## Component Design

### Navigation Bar

- **Background**: White with subtle shadow
- **Height**: 72px
- **Typography**: 16px, medium weight
- **Active State**: Orange background, white text
- **Hover State**: Subtle gray background transition

### Hero Section

- **Background**: Light gray (#F8F9FA)
- **Layout**: Split layout with text left, visuals right
- **Emphasis**: Large bold typography with color accents
- **CTA Buttons**: Primary orange, secondary white with border

### Cards & Panels

- **Background**: White
- **Border Radius**: 12px
- **Shadow**: Subtle drop shadow (0 4px 16px rgba(0,0,0,0.1))
- **Padding**: 24px
- **Hover State**: Slight shadow increase

### Data Visualization

- **Chart Colors**: Purple, orange, yellow gradients
- **Background**: Light backgrounds for contrast
- **Typography**: Dark text with clear hierarchy
- **Metrics**: Large bold numbers with context text

### Buttons

- **Primary**: Orange background, white text, rounded corners
- **Secondary**: White background, dark text, subtle border
- **Border Radius**: 8px
- **Padding**: 12px vertical, 24px horizontal
- **Hover**: Subtle color darkening

## Icon System

### Style

- **Type**: Outlined and filled icons
- **Size**: 24px standard, 16px small, 32px large
- **Color**: Inherits from parent or uses brand colors
- **Background**: Circular colored backgrounds for emphasis

### Usage

- **Emoji Integration**: Strategic use of emojis in headlines
- **Functional Icons**: Clear, recognizable symbols
- **Decorative Icons**: Subtle accent elements

## Data Visualization

### Chart Design

- **Colors**: Brand palette with good contrast
- **Typography**: Clear labels and numbers
- **Backgrounds**: Light colors for readability
- **Spacing**: Adequate padding around elements

### Metrics Display

- **Numbers**: Large, bold typography
- **Context**: Smaller descriptive text
- **Trends**: Color-coded indicators (green up, red down)
- **Grouping**: Clear visual hierarchy

## Interactive States

### Hover Effects

- **Buttons**: Subtle color darkening
- **Cards**: Shadow increase
- **Links**: Color change to orange
- **Icons**: Slight scale increase

### Focus States

- **Accessibility**: Clear focus indicators
- **Color**: Orange outline for consistency
- **Visibility**: High contrast for screen readers

## Responsive Design

### Breakpoints

- **Mobile**: 320px - 767px
- **Tablet**: 768px - 1023px
- **Desktop**: 1024px+

### Mobile Adaptations

- **Navigation**: Hamburger menu
- **Layout**: Single column stacking
- **Typography**: Reduced scale for smaller screens
- **Touch Targets**: Minimum 44px for buttons

## Accessibility Guidelines

### Color Contrast

- **Text**: Minimum 4.5:1 contrast ratio
- **Large Text**: Minimum 3:1 contrast ratio
- **Interactive Elements**: Clear visual feedback

### Typography

- **Hierarchy**: Clear heading structure (H1, H2, H3)
- **Readability**: Adequate line spacing and font sizes
- **Language**: Clear, concise copy

## Animation & Transitions

### Principles

- **Duration**: 200-300ms for UI transitions
- **Easing**: Smooth, natural motion curves
- **Purpose**: Enhance UX, not distract

### Common Animations

- **Hover States**: Subtle color/shadow changes
- **Loading States**: Progress indicators
- **State Changes**: Smooth transitions between states

## Content Strategy

### Voice & Tone

- **Professional**: Clear, authoritative
- **Friendly**: Approachable, human
- **Confident**: Strong value propositions

### Copy Guidelines

- **Headlines**: Action-oriented, benefit-focused
- **Body Text**: Concise, scannable
- **CTAs**: Clear, specific action words

## Best Practices

### Performance

- **Images**: Optimized formats and sizes
- **Icons**: SVG format for scalability
- **Loading**: Progressive enhancement

### Usability

- **Navigation**: Clear, consistent
- **Feedback**: Immediate response to actions
- **Error Handling**: Helpful, actionable messages

### Brand Consistency

- **Logo Usage**: Consistent placement and sizing
- **Color Application**: Systematic use of brand colors
- **Typography**: Consistent hierarchy and spacing

---

**Note**: This design system emphasizes clean, modern aesthetics with strong data visualization capabilities. The orange accent color should be used strategically to guide user attention and create visual hierarchy throughout the interface.
